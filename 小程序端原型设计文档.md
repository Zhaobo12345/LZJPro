# 小程序端项目模块 - 可交互原型设计文档

## 文档说明

本文档为小程序端项目模块的可交互原型设计文档，包含：
- **页面布局设计**：详细的页面结构和元素布局
- **交互设计说明**：可在Axure、墨刀等工具中实现的交互逻辑
- **配色方案**：数字化色调，简洁专业
- **设计规范**：统一的视觉和交互规范

### 2.3 交互说明

#### 2.3.1 页面加载

```
交互流程：
1. 页面加载时显示骨架屏
2. 数据加载完成后显示内容
3. 加载失败显示错误提示

骨架屏设计：
- 使用灰色占位块
- 添加微动画效果（闪烁）
- 保持页面结构一致
```

#### 2.3.2 下拉刷新

```
交互流程：
1. 用户下拉页面
2. 显示刷新指示器
3. 刷新数据
4. 隐藏刷新指示器

刷新指示器：
- 位置：顶部
- 样式：加载动画 + "刷新中..."
- 颜色：#1890FF
```

#### 2.3.3 项目卡片点击

```
交互流程：
1. 用户点击项目卡片
2. 卡片显示点击效果（背景色变化）
3. 跳转到项目详情页
4. 页面从右侧滑入

点击效果：
- 按下：背景色变为#F5F5F5
- 抬起：背景色恢复#FFFFFF
- 动画时长：200ms
```

#### 2.3.4 待办事项点击

```
交互流程：
1. 用户点击待办事项
2. 显示点击效果
3. 跳转到任务详情页
4. 页面从右侧滑入

点击效果：
- 按下：背景色变为#F5F5F5
- 抬起：背景色恢复#FFFFFF
```

#### 2.3.5 快捷入口点击

```
交互流程：
1. 用户点击快捷入口图标
2. 图标显示点击效果（缩放）
3. 跳转到对应功能页面

点击效果：
- 按下：缩放至0.95
- 抬起：缩放恢复1.0
- 动画时长：100ms
```

#### 2.3.6 查看全部点击

```
交互流程：
1. 用户点击"查看全部"
2. 跳转到对应列表页
3. 页面从右侧滑入
```

### 3.3 交互说明

#### 3.3.1 状态筛选

```
交互流程：
1. 用户点击状态选项卡
2. 选项卡显示选中效果（下划线）
3. 列表刷新，显示对应状态的项目

选中效果：
- 选中：文字颜色#1890FF，下划线#1890FF
- 未选中：文字颜色#595959，无下划线
- 动画：下划线滑动过渡，时长300ms
```

#### 3.3.2 搜索功能

```
交互流程：
1. 用户点击搜索框
2. 显示键盘
3. 用户输入搜索内容
4. 实时搜索并显示结果

搜索逻辑：
- 实时搜索：输入时实时搜索
- 防抖处理：输入停止300ms后搜索
- 搜索范围：项目名称、项目编码
- 高亮显示：搜索关键词高亮显示
```

#### 3.3.3 项目卡片交互

```
交互流程：
1. 用户点击项目卡片
2. 卡片显示点击效果
3. 跳转到项目详情页

左滑操作：
1. 用户左滑项目卡片
2. 显示操作按钮（编辑、删除）
3. 点击操作按钮执行对应操作

长按操作：
1. 用户长按项目卡片
2. 显示操作菜单（分享、复制、删除）
3. 点击菜单项执行对应操作
```

#### 3.3.4 上拉加载

```
交互流程：
1. 用户滚动到列表底部
2. 显示加载指示器
3. 加载下一页数据
4. 隐藏加载指示器

加载策略：
- 触发距离：距离底部100px
- 每页数量：10条
- 加载指示器：加载动画 + "加载中..."
```

#### 3.3.5 创建项目

```
交互流程：
1. 用户点击"创建项目"按钮
2. 跳转到项目创建页面
3. 页面从右侧滑入
```
